## What "TOJ feel" means here

This transcript is held to the same standard TOJ sells every client. It follows the
canon defined in `TojCampaign/agents/skills/`:

- **`toj-methodology.md`** — the five pillars, canonical order, 0–100 scoring bands, and
  the rule that no pillar scores above 40 without retrievable evidence.
- **`five-pillars.md`** — the score + one-line evidence + one-line gap output format.
- **`toj-voice.md`** — operator-not-audience register, banned-word list, em-dash
  discipline, curly quotes, and endings that name the next action.

House-style reference: `TojCampaign/public/sample-transcript-authority.html`.

## Structure (canonical)

1. **Executive summary** — one page, one decision, the one thing to fix first.
2. **Foundation Score** — five pillars scored, evidenced, gapped, each with a one-week
   move and a success signal. Radar + legend.
3. **Weakest pillar** — Revenue Architecture (48/100), the deep-dive.
4. **Cross-pillar move** — the alumni graph (Pylon Wall · State of the Pipeline · Alumni
   Membership); one asset that fixes two pillars.
5. **Two-week ship list** — named owner + numeric done-when per item.
6. **Six-week horizon** — the Foundation Playbook, adapted revenue-first.
